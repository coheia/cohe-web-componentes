## NPM Scripts

* **storybook**: Starts Storybook's development server.
* **icons**: Gerenates iconography based in a SVG icons Folder.
* **dev**: Build and watch concurrently `npm run build:css-to-ts --watch`, `vite build --watch` and `npm run build:elements-type --watch`.
* **build**: Runs `vite build` for production build, depends on `build:css-to-ts`, `build:elements-type` and `build:storybook`.
* **build:css-to-ts**: Finds all `.css` files and converts them to `.css.ts` files using `css-to-ts.js` script, depends on `build:sass`.
* **build:sass**: Compiles all `.scss` files to `.css` files.
* **build:storybook**: Compiles storybook into `.storybook-static`.
* **build:elements-type**: Analyzes component files using the glob pattern `src/components/**/swc-+([a-z]).ts` and generates a `custom-elements.json` file using the [**wca** CLI](https://www.npmjs.com/package/web-component-analyzer).
* **clean**: Exec `clean:dist` and `clean:storybook`.
* **clean:dist**: Removes the `dist` directory.
* **clean:storybook**: Removes the `storybook-static` directory.
* **clean:wireit-cache**: Removes the `.wireit` directory.
* **clean:gitignore**: Removes all files from repo that are in the `.gitignore` file
* **clean:all**: Removes the `node_modules`, `.storybook-static`, `dist`, `.wireit` directories and will remove all files from repo that are in the `.gitignore` file.
* **prepublish**: Prepare the `dist` folder to be published as a library.
* **publish:patch**: Runs `prepublish`, create a new version patch and publish at NPM.
* **publish:minor**: Runs `prepublish`, create a new version minor and publish at NPM.
* **publish:major**: Runs `prepublish`, create a new version major and publish at NPM.
